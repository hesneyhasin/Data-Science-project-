# Data Science Project

This repository contains data science projects implemented in Jupyter notebooks.

## Contents

### Notebooks

- **CSE303Project.ipynb** - Main project notebook for CSE303
- **ProjectCSE303.ipynb** - Additional project notebook for CSE303

## Getting Started

To run these notebooks, you'll need:

- Python 3.x
- Jupyter Notebook or JupyterLab
- Required Python packages (see notebook dependencies)

## Usage

1. Clone the repository
2. Install required dependencies
3. Open the notebooks in Jupyter:
   ```bash
   jupyter notebook CSE303Project.ipynb
   ```
   or
   ```bash
   jupyter notebook ProjectCSE303.ipynb
   ```
